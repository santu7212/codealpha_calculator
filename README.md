# 🧮 Basic Calculator

A simple and responsive calculator built using **HTML, CSS, and JavaScript**.  
This project performs basic arithmetic operations and includes keyboard support.

---

## 🚀 Features

- ➕ Addition
- ➖ Subtraction
- ✖ Multiplication
- ➗ Division
- 🧹 Clear button
- ⌨ Keyboard support
- 📱 Responsive design
- ⚠ Error handling for invalid expressions

---

## 🛠 Technologies Used

- HTML5
- CSS3 (Grid Layout)
- JavaScript (DOM Manipulation & Event Handling)

---

## 🎮 Keyboard Support

| Key | Function |
|------|----------|
| 0–9 | Enter numbers |
| + - * / | Operators |
| . | Decimal |
| Enter | Calculate result |
| Backspace | Delete last character |
| Escape | Clear display |
